# Objective

Improve a poorly written prompt by identifying ambiguity, missing constraints, and unclear instructions. Produce a revised version that is more specific, structured, and likely to generate high-quality AI responses.

---

# Original Prompt

> Write something about dogs.

---

# Prompt Analysis

## Problems Identified

### Issue 1 — Too Broad

The prompt provides no indication of what should be written.

Possible interpretations include:

- Essay
- Story
- Blog post
- Scientific explanation
- Social media caption

---

### Issue 2 — No Target Audience

The response could be intended for:

- Children
- Pet owners
- Veterinarians
- Researchers

Without an audience, response quality becomes inconsistent.

---

### Issue 3 — Missing Constraints

The prompt does not specify:

- Length
- Tone
- Style
- Format

---

### Issue 4 — No Objective

The model is not told *why* it is writing.

Examples:

- Inform
- Entertain
- Persuade
- Educate

Different objectives produce very different outputs.

---

# Improved Prompt

Write a 500-word beginner-friendly blog post explaining why dogs make good family pets.

Requirements:

- Use simple English.
- Include at least five benefits.
- Mention two responsibilities of dog ownership.
- Maintain a friendly and informative tone.
- Use headings and bullet points where appropriate.
- End with a short conclusion.

---

# Why This Prompt Is Better

| Improvement | Benefit |
|------------|---------|
| Defines audience | Produces more relevant language |
| Specifies objective | Guides the model toward the desired outcome |
| Adds length requirement | Produces consistent output size |
| Adds formatting | Improves readability |
| Specifies tone | Reduces variation |
| Includes constraints | Makes the response easier to evaluate |

---

# Comparison

## Improved Prompt

Produces a response that is:

- Focused

---

# Key Principles Demonstrated

- Reduce ambiguity
- Specify the objective
- Define the audience
- Add measurable constraints
- Specify formatting
- Specify tone
- Keep instructions concise

---

# Reflection

## What I Learned

Small improvements to prompt wording can significantly improve response quality. Clearly defining the audience, objective, constraints, and expected format reduces ambiguity and makes AI outputs more reliable and consistent.

---

## What I'd Do Differently

Experiment with multiple prompt versions to compare response quality and determine which constraints have the greatest impact on the final output.
